# MovieLens Databricks Data Engineering Project

## Project Overview

An end-to-end Data Engineering project built on the MovieLens dataset using Databricks, PySpark, Delta Lake, and Databricks AI Functions.

The project follows the Medallion Architecture pattern and demonstrates:

- Bronze Layer ingestion
- Silver Layer transformations
- SCD Type 2 implementation
- Gold Layer analytics
- GenAI-powered business insights
- Dashboard reporting

 ## Key Features

### Bronze Layer
- Raw MovieLens data ingestion

### Silver Layer
- Data cleansing and transformations
- Incremental processing

### SCD Type 2
- Historical tracking using:
  - start_date
  - end_date
  - current_status

### Gold Layer Analytics
- Top Genres
- Genre Trends by Release Year
- Average Ratings Analysis

### GenAI Layer
- AI-generated business insights
- Prompt-based narrative generation using Databricks AI Functions

### Dashboard
- Interactive visualizations
- Analytics combined with AI-generated commentary
